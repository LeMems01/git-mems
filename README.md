
# Santé Plus – Site Web Informatif sur les Maladies à Kinshasa

Un site web éducatif destiné à sensibiliser la population de Kinshasa sur les maladies courantes, leurs symptômes, modes de transmission, traitements, et méthodes de prévention.

---

## Objectif du projet

Santé Plus vise à fournir une plateforme simple, claire et accessible pour :
- Informer sur les maladies courantes à Kinshasa (paludisme, choléra, typhoïde, hépatite, etc.).
- Promouvoir des comportements préventifs grâce à des ressources fiables.
- Offrir un accès rapide aux contacts de centres de santé et d'institutions sanitaires.
- Permettre une navigation fluide sur mobile et ordinateur (responsive design).

## Fonctionnalités principales

- Interface claire et intuitive
- Navigation entre pages avec menu responsive
- Fiches informatives illustrées sur chaque maladie
- Page de contact avec formulaire + clic email / téléphone
- Intégration d’icônes et images médicales
- Page des centres de santé à Kinshasa
- Responsive design (PC, tablette, smartphone)
- Barre de recherche (fonctionnelle en JavaScript)

---

## Technologies utilisées

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript Vanilla (ES6)
- Icônes : Font Awesome
- Images libres : Pexels, Unsplash

## Utilisation

1. Cloner le projet :
   ```bash
   git clone https://github.com/votre-utilisateur/sante-plus.git
   ```
2. Ouvrir le dossier :
   ```bash
   cd sante-plus
   ```
3. Lancer le site :
   - Ouvrir `accueil.html` dans votre navigateur.
